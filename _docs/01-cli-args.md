---
title: "CLI arguments"
permalink: /docs/cli-args/
---

```
  -debug
        Log to ./debug.log
  -ff-profile string
        Firefox profile to use (default "default")
  -firefox string
        Path to Firefox executable (default "firefox")
  -http-server
        Run as an HTTP service
  -http-server-port string
        HTTP server address (default "4333")
  -startup-url string
        URL to launch at startup (default "https://google.com")
  -time-limit int
        Kill Browsh after the specified number of seconds
  -use-existing-ff
        Whether Browsh should launch Firefox or not
  -websocket-port string
        Web socket service address (default "3334")
  -with-gui
        Don't use headless Firefox
```
